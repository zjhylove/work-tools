## 常见问题

### 1. 连接问题

Q: SSH连接失败怎么办？
A: 检查网络连接，确认SSH服务器地址和端口是否正确，验证认证信息是否有效。

### 2. 性能问题

Q: 同时开启多个转发会影响性能吗？
A: 转发数量会占用系统资源，建议根据实际需求合理配置转发规则。

### 3. 安全问题

Q: 如何保护敏感配置信息？
A: 所有敏感信息都经过加密存储，建议定期更改密码和密钥。
